Геометријски редови

Геометријски редови су редови форме $$b+bq+bq^2+⋯+bq^{n-1}+⋯= \sum_{n=1}^\infty bq^{n-1}$$ у којима су a и r непроменљиви реални бројеви и a ≠ 0. Ови редови могу такође бити написани као \(\sum_{n=0}^\infty bq^{n}\). Количник q може бити позитиван, као у $$1+ \frac{1}{2}+ \frac{1}{4}+⋯+ (\frac{1}{2})^{n-1}+⋯,$$ или негативан, као у $$1- \frac{1}{3}+ \frac{1}{9}-…+ (\frac{1}{3})^{n-1}+⋯.$$ Ако је q = 1, n-ти парцијални збир геометријског реда је $$S_n=b+b(1)+b(1)^2+⋯+b(1)^{n-1}=nb,$$

Теорема 1. У геометријском низу (\(b_n\)) на првим чланом \(b_1\) и количником q, за све природне бројеве n важи: $$b_n= b_1•q^{n-1}$$ Доказ: Доказ се изводи примером математичке индукције. За n = 1 имамо \(b_1=b_1 • q^0\), па је тврђење тачно. Претпоставимо да важи \(b_n= b_1•q^{n-1}\) за неки природни број n. Тада имамо: $$b_{n+1}= b_н•q= b_1 • q^{n-1} •q= b_1 • q^n,$$ па је тврђење тачно за n + 1, што значи да је на основу принципа математичке индукције оно тачно за све природне бројеве. У многим задацима, поготово у применама математике, јавља се потреба да се израчуна збир првих n чланова геометријског низа.

Теорема 2. Збир првих n чланова геометријског низа (\(b_n\)) чији је количник q ≠ 0 износи $$S_n= b_1 \frac{q^{n-1}}{q-1}$$ Доказ: Како је \(b_1 q= b_2,b_2 q= b_3,…,b_n q=b_{n+1}\), следи да је $$S_n q= b_1 q+ b_2 q+⋯+ b_n q= b_2+ b_3+⋯+b_{n+1}.$$ Када од ове једнакости одузмемо тражени збир \(S_n= b_1+ b_2+⋯+ b_n\), добијемо $$S_n q- S_n= b_{n+1}- b_1= b_1 q^n- b_1,$$ односно $$S_n (q-1)= b_1 (q^n-1),$$ одакле следи истинитост тврђења.

ПРИМЕР 1 Индекс почиње са n = 1 Геометријски ред за b = 1/9 I q = 1/3 је $$\frac{1}{9}+ \frac{1}{27}+ \frac{1}{81}+⋯= \sum_{n=1}^\infty \frac{1}{9} (\frac{1}{3})^{n-1}=$$ $$= \frac{\frac{1}{9}}{1-\frac{1}{3}}= \frac{1}{6}.$$ ПРИМЕР 2 Индекс почиње са n = 0 Ред $$\sum_{n=0}^\infty \frac{(-1)^n 5}{4^n} =5- \frac{5}{4}+ \frac{5}{16}- \frac{5}{64}+⋯$$ Је геометријски ред са b = 5 и q = 1/4.

ПРИМЕР 3 Лоптра која одскаче Лопта је бачена са a метара изнад равне површине. Сваки пут лопта удари о површину и пошто падне на растојању h, она одскочи до растојања rh, где је r позитивно, али је мање од 1. Наћи укупно растојање које је лопта прешла навише и наниже.

Решење Укупно расојање је $$s=a+2ar+2ar^2+2ar^3+⋯=a+ \frac{2ar}{1-r}=a \frac{1+r}{1-r}$$ Ако је, на пример, a = 6m I r = 2/3, растојање је $$s=6 \frac{1+ \frac{2}{3}}{1- \frac{2}{3}}=6 \frac{\frac{5}{3}}{\frac{1}{3}}=30m$$ ПРИМЕР 4 Бесконачни децимални бројеви чије се цифре понављају Изразити бесконачан децимални број чије се цифре понављају 5,232323... као однос два цела броја.

Решење $$5.232323…=5+ \frac{23}{100}+ \frac{23}{100^2} + \frac{23}{100^3} +⋯=$$ $$=5+ \frac{23}{100} (1+ \frac{1}{100}+ \frac{23}{100^2}+⋯)=$$ $$=5+ \frac{23}{100} (\frac{1}{0.99})=5+ \frac{23}{99}= \frac{518}{99}$$ $$b = 1$$ $$q = \frac{1}{100}$$
